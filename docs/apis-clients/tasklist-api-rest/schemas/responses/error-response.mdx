### Fields

#### [<code style={{ fontWeight: 'normal' }}>ErrorResponse.<b>status</b></code>](#)<Bullet />`int` <Badge class="secondary" text="non-null"/>

> An integer that represents the HTTP status code of the error response. For example, 400 indicates a "Bad Request" error, 404 indicates a "Not Found" error, and so on.

#### [<code style={{ fontWeight: 'normal' }}>ErrorResponse.<b>message</b></code>](#)<Bullet />`string` <Badge class="secondary" text="non-null"/>

> A string that provides a brief description of the error that occurred.

#### [<code style={{ fontWeight: 'normal' }}>ErrorResponse.<b>instance</b></code>](#)<Bullet />`string`

> UUID for look up (eg. in log messages).
