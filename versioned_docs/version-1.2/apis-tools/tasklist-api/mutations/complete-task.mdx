Complete a task with taskId and optional variables. Returns the task.

```graphql
completeTask(
  taskId: String!
  variables: [VariableInput!]!
): Task!

```

### Arguments

#### `taskId` ([`String!`](../scalars/string.mdx))

#### `variables` ([`[VariableInput!]!`](../inputs/variable-input.mdx))

### Type

#### [`Task`](../objects/task.mdx)

Describes the User task.
